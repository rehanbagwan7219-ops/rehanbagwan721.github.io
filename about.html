What We Have Built — Group Project

As part of my B.Tech first year, our team developed a Smart Waste Classification System using Artificial Intelligence, Arduino, and MIT App Inventor.


AI-Based Waste Detection

Our team used Teachable Machine to train an AI model that classifies waste as Dry Waste or Wet Waste using the phone camera in real time.


MIT App Inventor Mobile App

We built a mobile app that shows DRY DETECTED or WET DETECTED on screen, announces it using Text-to-Speech, and sends a signal (D or W) to Arduino via Bluetooth.


Arduino + Bluetooth Hardware

We connected an Arduino UNO with an HC-05 Bluetooth module that receives the signal from the app and controls a physical waste sorting mechanism.


Block-Based Programming

Our team wrote the entire app logic using MIT App Inventor blocks — handling AI classification results, Bluetooth connection, sending text commands, and voice output without writing traditional code.
