My Projects

Design Thinking Project – Dice Game for Children

This project was developed using the principles of Design Thinking to create a simple and engaging dice game for children. The goal of the project was to make learning fun while improving problem-solving and logical thinking skills in young students.

Through this project, I learned how to understand user needs, create simple solutions, and test ideas to improve the final product.

AI for All Project – Scratch Game with AI Integration

In this project, I created a game using Scratch programming to teach basic coding concepts along with the introduction of Artificial Intelligence.

The game demonstrated how AI can be used to make programs interactive and intelligent. This project helped me understand the practical applications of AI in education and technology.

Chemistry Project – Eco-Friendly Transport Using Electric Buses

This project focused on promoting environmentally friendly transportation by using electric buses instead of fuel-based vehicles.

The objective was to reduce air pollution, save energy, and support sustainable development. This project helped me understand the importance of green technology and eco-friendly solutions in modern society.
